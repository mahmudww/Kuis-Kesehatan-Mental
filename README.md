# Mental Wellness Quiz

Kuis ini dibuat untuk membantu pengguna Gen Z/Milenial mengecek kondisi mental mereka dalam 4 area: Depresi, Kecemasan, Stres, dan Kelelahan (Burnout).

Kuis ini hanya membutuhkan waktu kurang dari 3 menit.

Penting: Kuis ini hanya alat bantu self-awareness dan bukan pengganti diagnosa profesional.
